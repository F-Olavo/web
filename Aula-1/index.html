Curso: Programador Web

O curso de programção web tem por objetivo a capacitação de profissionais para desenvolvimento de sistemas web atuando em programação front-end ou back-end. O curso tem carga hóraria de 240 horas e sua grade é composta por conteúdos de Front-end(HTML, CSS e JavaScript) e Back-end (PHP e SQL).

O senac conta com diversoso cursos na área de programação com: Lógica de programação, programação de sistemas, fulllstack, formação em python
